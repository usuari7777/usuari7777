Programación en Python

Esta web es un curso sobre programación en lenguaje Python. Explicaremos los conceptos básicos del lenguaje, suponiendo una base previa de programación, sobre qué es una variable, una condición, un bucle, etc., para así tratar estos conceptos aplicados al lenguaje Python. En caso de no tener esa base, quizá sea conveniente adquirirla echando un vistazo a otras webs como esta, donde se dan nociones de programación en otro lenguaje, como es C#.
