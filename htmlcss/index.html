Diseño web con HTML y CSS

Esta web es un curso sobre diseño web utilizando HTML y CSS. Entre otros contenidos, se explica la estructura de los documentos HTML o xHTML, las etiquetas principales que se pueden utilizar para estructurar la información de las páginas, y también cómo dotarlas de diferentes estilos utilizando hojas CSS. Finalmente, se tienen unas nociones de diseño web adaptativo y uso de frameworks de diseño.
