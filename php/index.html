Desarrollo web con PHP

Esta web es un curso sobre desarrollo web en servidor utilizando el lenguaje PHP. Explicaremos los conceptos básicos del lenguaje, suponiendo una base previa de programación, sobre cómo definir variables, condiciones, bucles, funciones, etc. Después, pasaremos a explicar cómo utilizar el lenguaje combinado con la parte del cliente (HTML y JavaScript), cómo procesar peticiones o envíos de formularios, cómo modularizar el código en varios ficheros, conectar a bases de datos, etc. También veremos algunas nociones de los frameworks PHP, y las ventajas que supone utilizarlos.
