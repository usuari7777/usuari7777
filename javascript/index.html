Desarrollo web con JavaScript

Esta web es un curso sobre desarrollo web en cliente utilizando el lenguaje JavaScript. Explicaremos los conceptos básicos del lenguaje, suponiendo una base previa de programación, sobre cómo definir variables, condiciones, bucles, funciones, etc. Después, pasaremos a analizar algunos de los usos más habituales del lenguaje: explorar los contenidos de una página para buscar información o modificarla, responder a ciertos eventos en la página, etc. También veremos cómo emplear algunas librerías externas, con algunos ejemplos sencillos.
